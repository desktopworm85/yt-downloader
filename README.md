# yt-downloader

This is a youtube downloader created by desktopworm85. It provides a text box for the user to paste a youtube video/playlist url and select mp3 or mp4 file download.
Once selected, press download and it'll download the video in the same directory as yt-downloader.exe.

Installation instructions:
Download the [latest release](https://github.com/desktopworm85/yt-downloader/releases) zip file.
Extract the .ZIP file anywhere and run yt-downloader.exe. It will automatically update the file so there is no need to manually run the updater.
